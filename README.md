## Overview

**End Reforged** is a Fabric mod that expands and enhances **The End** dimension in Minecraft. Explore new biomes, encounter unique mobs, and discover exciting structures while keeping the vanilla feel intact. This mod is built for **Fabric Loader** and is compatible with Fabric API.

### Features

- Adds new biomes, terrain features, and structures in The End.
- Introduces new mobs and challenges.
- Includes unique loot and rare drops.
- Fully compatible with Fabric mods and mod loaders.
- Configurable options for customizing gameplay.

### Installation

1. Install **Fabric Loader**: [https://fabricmc.net/use/](https://fabricmc.net/use/)
2. Install **Fabric API**: [https://www.curseforge.com/minecraft/mc-mods/fabric-api](https://www.curseforge.com/minecraft/mc-mods/fabric-api)
3. Download the latest `End Reforged` jar from the [releases](#) page.
4. Place the jar file in your Minecraft `mods` folder.
5. Launch Minecraft with the Fabric profile.
